# Web Viewer

这个 VSCode 插件允许用户输入链接并在 VSCode 窗口中打开预览。

## 功能

- 支持用户输入链接
- 在 VSCode 窗口中打开链接预览

## 安装

1. 打开 VSCode
2. 打开命令面板 (Cmd+Shift+P 或 Ctrl+Shift+P)
3. 输入 "扩展: 从 vscode 市场安装"
4. 搜索 "Web Viewer"
5. 点击 "安装"

## 使用方法

1. 打开命令面板 (Cmd+Shift+P 或 Ctrl+Shift+P)
2. 输入 "预览网页"
3. 在弹出的输入框中输入要预览的链接
4. 链接内容将在 VSCode 窗口中打开预览

## 要求

VSCode 1.94.0 或更高版本
